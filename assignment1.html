by David Magnuson

Mission

Provide a visual interface to support the school of engineering with student advising and academic planning

Goals

  1. Fully accessible
  2. Support dark mode
  3. Responsive interface

Inspiration

  • I chose this UI mockup design because I could not find a website that had this type of design.
  • I chose this design because it is simplistic and modern looking while also being able to be highly customized and responsive when desired.
  • I enjoy this type of layout because you are able to view all your tabs/pages easily while you can view all the content of the current page.

Pages

Landing

This page will be the first page seen by the user. It will contain a welcome message as well as a button to login. If the user is already logged in they will be directed to their dashboard, if they need to login they will be directed to microsoft active directory to authenticate themselves.

Dashboard

This will be the first page seen by an aurthenticated user. It will contain information about the user and breif summarys of the rest of their pages.

Courses

This page will contain a list of all the courses that are offered by the users program. The user will be able to search and filter for courses as they desire.

Course

This page will contain all the information about a certain course.

Plans

This page will have all of the users academic plans and be able to quickly view the status of each of them.

Plan

This page will have all the information about a specific users plan. The user will be able to view their program progress and edit their academic plan as needed.

Students

This page will have all of the advisors students and only be accessable by the advisor.

Student

This page will have all the information about a specific student and their academic plans.

Terms

This page will have all of the terms that have happened already as well as terms that the advisor is activley planning. This page will only be visable to the advisor.

Term

This page will have all the information about a specific term and allow the advisor to edit the terms courses based on their users academic plans. This page will only be visable to the advisor.

Audience

The audience for my site are the students at advisors and Dunwoody, specifically those in the School of Engineering
The audience is seeking a way to easily and visably view a students academic plan, make multiple if needed and edit & change them to customize their plan for graduation
